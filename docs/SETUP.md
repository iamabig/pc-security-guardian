# 🛠️ 개발 환경 설정 가이드 (설계 단계 버전)

이 문서는 **코드가 아직 없을 때** 레포를 어떻게 클론하고, 이후 Phase 1에서 무엇을 할지 안내하는 설계용 문서입니다.

## 1. 레포 클론
```bash
git clone https://github.com/iamabig/pc-security-guardian.git
cd pc-security-guardian
```

## 2. 다음 단계 (Phase 1에서 할 일)
- `npx create-next-app`으로 Next.js 프로젝트 초기화
- TailwindCSS, ESLint, Prettier 설정
- `src/`, `api/`, `tests/` 폴더 실제 코드로 채우기

지금은 **설계 문서만 있는 상태**이고, 이 문서를 기준으로 개발을 시작하면 됩니다.
